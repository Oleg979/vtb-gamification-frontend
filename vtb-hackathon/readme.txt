Как запустить проект

1. Перейти в папку backend
2. В консоли выполнить команду npm i && npm run start:dev
3. Перейти в папку frontend
4. Открыть index.html
4. Для открытия ЛК сотрудника Марии(для сценария ниже) дописать в адресной строке ?userId=4

Архитектура проекта
1. Blockhain API, предоставленный ВТБ
2. Gamification BE, лежит в папке backend
3. Gamification UI, лежит в папке frontend
4. База данных Postgres, развернутая в облаке Amazon

Основной бизнес-сценарий: комплексная демонстрация механик геймификации(был продемонстрирован на чекпоинте)
Имеем двух юзеров, Ивана и Марию.
Иван владеет редким NFT, которое в данный момент оценивается в 10к цифровых рублей.
Мария выполняет поставленную в личном кабинете цель и тоже получает в награду экземпляр этого NFT. В результате его стоимость
в системе автоматически регулирется и падает до 9900 цифровых рублей.
Иван заходит в маркетплейс и хочет купить ноутбук. Но у него всего 1000 ЦР на счету. Поэтому он решает продать свой 
NFT и обменять его на деньги. В результате он покупает ноутбук, в результате чего в системе автоматически срабатывает событие получения
им достижения "Купить товар на 10к цифровых рублей", которое отображается в его ЛК. Цена NFT вновь поднимается до 10к из-за уменьшения
количества владельцев.
